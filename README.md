# ddys-crm-admin

## 简介

**ddys-crm-admin** 项目是基于 react+react-router+antd 框架开发的后台管理系统


## 如何开始
````````````````````
yarn

yarn start
````````````````````

````````````````````
router 配置 src/routers/router
````````````````````
### 目录结构
<pre>
├── config                      // 项目编译配置
├── src                         // 生产目录
│   ├── components              // 组件目录
│   ├── pages     // 页面目录

</pre>
